# GymPath - 開発概要

## コンセプト

「ジム初心者が迷わず筋トレできるアプリ」

AIがメインではなく、

- 店舗情報
- マシン情報
- 初心者向け導線
- 実際のジム体験

をベースに、
初心者が安心してジムに通える体験を作る。

---

# 解決したい課題

ジム初心者は、

- 何をすればいいかわからない
- どのマシンを使うかわからない
- マシンの場所がわからない
- 使い方がわからない
- 現実的なメニューが組めない

という問題を抱えている。

GymPathでは、
「その店舗に存在するマシン」を元に、
初心者向けメニューを提供する。

---

# MVP（最初に作る機能）

まずはシンプルに作る。

## 実装予定

- ログイン
- 店舗選択
